# My Portfolio
This is a portfolio I made to showcase my projects and skills, some info about me, and contacts.

I made some cool little animations such as the terminal and the blurred background circle following the cursor.

## Installing
```bash
#clone the repo
git clone https://github.com/ivg1/portfolio.git
cd portfolio

#install dependencies
npm install
```

## Hosting
To run the development server, run ```npm run dev```, and then follow the link the terminal outputs.

To build files for production, run ```npm run build```. The build files are in the ```src/dist``` folder. Copy them where you need to.
